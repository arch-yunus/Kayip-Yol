# Hazar Mirası ve Kayıp On Kabile

Hazar İmparatorluğu'nun dağılması sonrası Kafkasya'nın batısına çekilen toplulukların ve Asur sürgünü sonrası bölgeye yerleşen "Kayıp On Kabile" teorilerinin Acaristan coğrafyasındaki karşılıkları bu bölümde araştırılır.

## Araştırma Konuları
- **Hazar Göç Yolları:** Batı Kafkasya'ya sığınan Hazar boyları.
- **On Kabile Teorisi:** Bölge halklarındaki tek tanrılı inanç eğilimlerinin tarihsel kökeni.
- **Genetik ve Kültürel Entegrasyon:** Yerel Türk boyları ile Hazar mirasının birleşimi.
